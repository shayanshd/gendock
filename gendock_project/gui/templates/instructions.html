Application Instructions

Upload

  • Select a CSV dataset for uploading.
  • Ability to upload multiple CSV files simultaneously. Each file will appear under the "Uploaded CSV File" section.
  • Ensure each CSV file contains a 'smiles' column in lowercase.
  • Select multiple datasets as needed for combination and processing. Selected datasets will be prepared for cleaning, followed by training.
  • Option available to cancel the cleaning process at any stage.

Train

  • Displays all datasets that have undergone cleaning.
  • Choose a dataset for model training.
  • Specify the number of epochs for the training process.
  • Option to interrupt and cancel the training process if needed.
  • Training weights will be saved automatically upon cancellation or completion.

Generate

  • Input the number of compounds you wish to generate.
  • Specify the desired length for the number of smiles used in docking.
  • Ensure the sample number is greater than the desired length; neither can be less than 3.
  • Once the initial generation is complete and the validity of compounds is confirmed, you can proceed with the docking process.
  • Prior to docking, configure the receptor: choose a Receptor pdbqt file and set the Center and Size for X, Y, and Z axes.
  • Initiate the docking process, keeping an eye on the generation number.
  • Upon completion of docking, two tables will be presented: one for the current generation results and another for all generations.
